# Дипломный проект: Home Credit Default Risk

## Общее описание проекта
Данный проект выполнен в рамках дипломной работы курса Data Science от SkillFactory и посвящён решению реальной задачи кредитного скоринга на основе данных соревнования **Home Credit Default Risk** (Kaggle).  
Цель проекта — построить надёжную, интерпретируемую модель машинного обучения, способную предсказывать вероятность возникновения у заёмщика трудностей с погашением кредита.

Проект демонстрирует **полный цикл Data Science**, включая анализ данных, очистку, генерацию признаков, построение моделей, валидацию, интерпретацию и подготовку решения к продакшену.

---

## Цель проекта
- Применить весь спектр навыков, полученных в ходе обучения на курсе SkillFactory DS.  
- Построить качественную, воспроизводимую ML‑модель для оценки кредитного риска.  
- Продемонстрировать умение работать с **крупными, многотабличными, разнородными данными**.  
- Подготовить решение, которое может быть использовано в реальной финансовой организации.

---

## Бизнес‑контекст
Финансовые компании ежедневно принимают решения о выдаче кредитов. Ошибки в оценке платёжеспособности клиентов приводят к:

- финансовым потерям,
- ухудшению качества кредитного портфеля,
- снижению прибыли.

Модель кредитного скоринга позволяет:

- уменьшить долю дефолтов,
- повысить точность одобрения заявок,
- автоматизировать процесс принятия решений,
- поддерживать риск‑ориентированное ценообразование.

В рамках проекта предсказывается переменная **TARGET**, где:

- `1` — клиент с высокой вероятностью возникновения проблем с погашением,
- `0` — клиент надёжен.

## Этапы разработки проекта

### **1. Анализ структуры данных**
- Изучение всех таблиц  
- Определение ключей (`SK_ID_CURR`, `SK_ID_PREV`, `SK_ID_BUREAU`)  
- Построение схемы связей  
- Поиск потенциальных утечек целевой переменной  

### **2. Исследовательский анализ данных (EDA)**
- Анализ распределений  
- Выявление пропусков и выбросов  
- Корреляционный анализ  
- Сравнение признаков для TARGET=0/1  
- Статистические тесты: t‑test, U‑test, χ², KS‑test, ANOVA  

### **3. Очистка данных**
- Обработка пропусков  
- Коррекция категориальных значений  
- Удаление невозможных значений  
- Преобразование временных признаков  
- Обработка аномалий  

### **4. Генерация признаков**
- Агрегации по всем дополнительным таблицам  
- Создание доменных признаков (отношения, коэффициенты, индексы)  
- Кодирование категориальных переменных  
- Отбор признаков (SHAP, permutation importance, MI, статистические тесты)  

### **5. Базовые модели**
- Логистическая регрессия  
- Дерево решений  
- Простые эвристики  
- Установка baseline‑метрики  

### **6. Продвинутые модели**
- LightGBM  
- CatBoost  
- XGBoost  
- Random Forest / Gradient Boosting  
- Stacking  
- Подбор гиперпараметров (Optuna)  
- Борьба с дисбалансом классов  

### **7. Валидация и эксперименты**
- Stratified K‑Fold  
- Проверка на утечки  
- Фиксация случайных сидов  
- Логирование экспериментов  

### **8. Интерпретация модели**
- SHAP  
- Важность признаков  
- Частичные зависимости  
- Бизнес‑интерпретация ключевых факторов риска  

### **9. Подготовка к продакшену**
- Скрипт обучения модели  
- Скрипт предсказаний  
- Сериализация пайплайна  
- Пример использования  

### **10. Финальный отчёт и презентация**
- Итоги анализа  
- Сравнение моделей  
- Выводы и рекомендации  
- Материалы для защиты
